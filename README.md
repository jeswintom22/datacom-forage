# Kudos System Repository

This repository is scaffolded for the internal Kudos feature described in [SPECIFICATION.md](SPECIFICATION.md).

## Planned Structure

- `apps/web` - React frontend for the employee portal
- `apps/api` - REST API for kudos creation, feed retrieval, and moderation
- `packages/shared` - Shared types, validation helpers, and utilities
- `docs` - Supporting design notes and implementation references

## Current State

The repository is intentionally minimal at this stage. It is ready for implementation work to begin from the approved specification.
